__<h2>탤런트뱅크 기업 과제</h2>__

```
주제 : 탤런트뱅크의 아웃바운드 마케팅 데이터

목적 : 추후 어떤 분야의 인재를 영입해야하며, 어떤 프로젝트를 유치해야하는 지에 대한 Insight 제시
```

```
1. 기업의 수요 조사 : 기업이 어떤 일(프로젝트)에 탤런트뱅크 서비스를 이용할 것인가

2. 전문가 분포 조사 : 어떤 인재를 어디서 영입할 것인가

3. 데이터 비교 통계 : 수집된 자료들을 비교 통계

4. 통계 결과 및 제언
```

```
Crawler : 사이트별 공고를 수집하기 위한 크롤러
Preprocessing : 수집한 자료들 전처리에 도움받으려고 만듬
Excel Data : 수집된 자료들을 전처리 과정을 통해 종합함
PowerBI : 수집된 자료를 바탕으로 시각화
```

```
TB_1_PPT.pdf : 발표용 PPT
TB_2_PB.pdf : 발표 보조용 Power BI
```
